# 🚀 Roadmap

> 以下是一些可以优化的地方或新功能，仅作记录之用，暂时没有开发计划。

## 🐛 修复

- 【重要】自动刷新小米账号登录凭证
  - 小米账号登录凭证有效期 1 个月，到期后需要自动刷新（mi-service-lite）

## 💪 优化

- 自动识别设备型号
  - 通过查询设备 miot spec 文件，自动获取指令参数
  - 自动识别设备属性值是否有读取权限
- 添加镜像更新说明
  - 添加 db 文件导入/出教程，用于备份恢复对话历史记录

## ✨ 新功能

- 增强对话系统
  - 支持自定义对话系统 Prompt 模板
  - 添加是否启用对话模式的开关
  - 支持通过语音命令清除上下文
- MioT AI Agents
  - 支持小爱音箱控制米家设备
  - 通过 Agent 机制自动调用合适的工具（设备）
- RAG
  - wikis embedding
  - memory embedding
- 插件系统
  - 自定义语音指令
  - 联网查查询最新数据
